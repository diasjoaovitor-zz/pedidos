# Pedidos

Sistema para controle de pedidos em pequenas empresas

## Tecnologias

- [ReactJS](https://pt-br.reactjs.org/)
- [Firebase](https://firebase.google.com/?hl=pt)
- [MUI](https://mui.com/)

## Como rodar o projeto

```
git clone https://github.com/diasjoaovitor/pedidos.git
cd pedidos
yarn
yarn dev
```
